# Venyl — GitHub Pages Ready

В проект уже встроены 3 mp3-файла в папке `music/`.

## Что делать дальше
1. Загрузи эту папку целиком в новый репозиторий GitHub.
2. Открой `Settings -> Pages`.
3. Выбери `Deploy from a branch`.
4. Выбери ветку `main` и папку `/root`.
5. Сохрани и дождись ссылки на сайт.

## В проекте уже есть
- `index.html`
- `assets/venyl_logo.png`
- `music/track1.mp3`
- `music/track2.mp3`
- `music/track3.mp3`

Если захочешь заменить музыку, просто замени mp3-файлы в папке `music/` и при необходимости поправь массив `tracks` внутри `index.html`.
